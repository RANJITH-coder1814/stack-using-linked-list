# 📚 Stack using Linked List (C++)

This project implements a **Stack** data structure using a **Linked List** in C++.

---

## 📌 Concept

A Stack follows the **LIFO (Last In First Out)** principle:

* The last element added is the first to be removed.

---

## 🚀 Operations Implemented

* ✅ Push (Insert element)
* ✅ Pop (Remove element)
* ✅ Peek (View top element)
* ✅ isEmpty (Check if stack is empty)

---

## 🧠 Approach

* Use a pointer `top` to track the top node
* Insert and delete from the beginning of the linked list

---

## ⏱️ Time Complexity

| Operation | Complexity |
| --------- | ---------- |
| Push      | O(1)       |
| Pop       | O(1)       |
| Peek      | O(1)       |
